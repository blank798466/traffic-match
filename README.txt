1.利用评价函数以及对应的时间差和重量差的平均值及其方差求得入口出口优先级二维数组
2.利用GS婚姻匹配算法进行匹配得出五轴入站出站匹配数据集
3.比较直接匹配和GS匹配

Input：时间差和重量差的平均值及其方差的经验值，五轴入口出口车辆数据
Output：五轴入站出站匹配数据（GS算法）

csvdata.py  # 获取csv中五轴车入口出口的时间序列数据和重量序列数据
diffdata.py  # 获取入站出站时间差数组和重量差数组以及序号序列数据
evlfunc.py  # 获取入站和出站的评价函数，获取入站出站的优先级数组
connectingcar.py  # 获取匹配车辆，观察区别
main.py  # 主运行函数
